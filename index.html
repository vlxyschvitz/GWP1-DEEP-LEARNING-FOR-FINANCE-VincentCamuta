CNN with Gramian Angular Fields

Image-based time series encoding
Each 60-day return window is transformed into a 60×60 Gramian Angular Summation Field image. A convolutional neural network then learns spatial patterns from these temporal fingerprints.
GAF Transformation
Simulated GAF · SPY sample
Method: Summation GAF (GASF)
Image size: 60×60 pixels
Encoding: Scaled returns → polar coordinates → angular addition
Why it works: Preserves temporal order while encoding correlations between time steps as spatial patterns learnable by convolution filters.

Multi-Output LSTM

Joint prediction across all asset classes
